## Figma 반입 옵션

### Option A — Standard Mode (권장)

React/Next.js 코드를 Figma Dev Mode로 공유:
1. 개발 서버 실행 — `pnpm dev` → `http://localhost:3000`
2. Figma Dev Mode에서 "React component" 플러그인 설치
3. 프로젝트의 `components/` 디렉터리를 연결하면 각 컴포넌트가 Figma 라이브러리로 자동 매핑됨
4. 디자인 토큰(`app/globals.css` CSS 변수)은 Figma Variables로 동기화

### Option B — HTML to Figma

각 페이지의 렌더링된 HTML을 Figma로 반입:
1. Chrome 확장 "html.to.design" 설치
2. 실행 중인 페이지에서 확장 실행 → Figma 캔버스로 전체 페이지 가져오기
3. `stitch-output/prompts.md`의 각 화면 프롬프트를 Figma AI에 전달 → 디자인 variant 생성

## 디자인 원칙 요약

- **Style**: Swiss Modernism 2.0 + Minimalism (12열 그리드, 수학적 간격 8px base)
- **데이터 밀도**: row 44px, padding 8/12px, 숫자는 JetBrains Mono + tabular-nums
- **색상**: 뉴트럴 slate + 단일 CTA sky-700. 상태는 4색 (info/success/warning/danger)
- **모션 다이얼 3**: 150–300ms ease-out-expo. `prefers-reduced-motion` 준수
- **Variance 다이얼 1**: 표준 레이아웃 (사이드바 + 메인 + optional 오른쪽 패널)
- **Density 다이얼 8**: 높은 정보 밀도 (대시보드/어드민 관례)
